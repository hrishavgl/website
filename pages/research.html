The objective of this research

The point of this research is to compare various elements of my website and their functinalities with that of other websites. This is in an effort to show my influences while designing my website as well as to demonstrate how it holds up with some other popular websites.

Comparision 1: Navbar Layout and Selection

The navbar for this website was influenced by joeabercrombie.com which is the website of one of my all-time favourite authors. I've gone for a similar navigation bar where the currently selected link has a red underline and my navbar has a fixed posittion at the top of the webpage. And I've also added a hover effect where the link colour turns red when the cursor is pointed over it.

See image of mentioned website

Comparison 2: Navbar Hamburger Menu

The navigation bar is fully responsive in that when it is minimized to a point where the navigation links don't have any space, they are replaced by a humburger menu. This menu, when pressesd, is degigned to show the navigation links in a vertical layout. This allows the navbar to be fully responsive and usable in screens of any aspect ratio.

See image of mentioned website

Comparison 3: Responsive UI Elements

Taking an example of YouTube, the videos are shown as rectangular box elements. You can generally see 4 videos horizontally. But if you minimize the width to 50%, you will see only 2 videos and the next two will be shifted to the row below. This is implemented in my website where there is the presence of multiple such UI elements horizontally next to each other.

Comparison 4: Responsive Text

Let's take Wikipedia as an example, almost everyone is familiar with it. When you minimize the webpage, the text present in the articles get shifted down to fit the screen. This feature is present is my website as well in order to maximize readability of text on small screens.

Comparison 5: Form Validation

Any form submitted should be validated to check if the user has filled in all the required input options. Otherwise, a message should be displayed accordingly. In this example, we see that the feedback form on Apple Inc's website shows a textbox with error message. I have taken a different approach where an alert box is displayed telling the user that the form fields are empty.
